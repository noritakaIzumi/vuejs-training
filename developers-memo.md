# Developers memo

## Init repository

Init package.json

```shell
npm init -y
```

edit `package.json`

- author
- license

Pin node.js version via Volta

```shell
volta pin node@18.17.1
```

```text
success: pinned node@18.17.1 (with npm@9.6.7) in package.json
```
